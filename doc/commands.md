# Lvim

To see the full warning/error messages: `gl`
Help on tree: `g?`
Switch buffer: H/L 
See all mapping: `:map`
Quit: `<space>q`

prev/next buffer: `H/L`

# Bookmarks
space down/up: navigate bookmarks in file
space B: toggle bookmark
space right/left: navigate bookmarks in files
space F: bookmark files
